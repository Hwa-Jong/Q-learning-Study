# Q-learning-Study
Q-learning Study

## Requirements
- Python 3.x
- numpy
- torch
- tqdm


## Usage
1. **Run the Q-learning:**
    ```bash
    python q_learning_algorithm.py
    ```
    or
    
    ```bash
    python q_learning_ai.py
    ```
